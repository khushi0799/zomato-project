Description:

This project analyzes Zomato restaurant data to uncover insights into customer preferences, cuisines, ratings, pricing, and geographic distribution. The interactive Power BI dashboard presents a comprehensive view of the global food service landscape.

It highlights restaurant distribution across countries and cities, top cuisines, average ratings, price ranges, and trends over time. The dashboard enables users to explore data dynamically using filters for city, country, price range, and time period.

Key Highlights:

Restaurant count by country and year

Top cuisines and customer preferences

Average ratings by country

City-wise and price-range analysis

Interactive maps and trend visuals

Tools & Technologies:
Power BI, Excel, Tableau, SQl, Data Modeling, Data Visualization
